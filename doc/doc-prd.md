# 📄 PRODUCT REQUIREMENT DOCUMENT (PRD)

## Project Name: Smart Job Portal (MERN Stack)

## Version: 1.0

## Year: 2026

---

# 1. 📌 Project Overview

Smart Job Portal ek modern web application hai jahan:

* Job seekers jobs search aur apply kar sakte hain
* Recruiters jobs post kar sakte hain
* Admin ke paas full control hota hai system ka

Ye project scalable, secure aur real-world use ke liye design kiya gaya hai.

---

# 2. 🎯 Objectives

* Job search process ko simple aur fast banana
* Recruiters ko best candidates provide karna
* Admin ko full system control dena
* AI-based smart features add karna

---

# 3. 👥 User Roles

## 3.1 Job Seeker (User)

* Register / Login
* Profile create/update
* Jobs search karna
* Apply karna
* Resume upload
* Chat with recruiter

## 3.2 Recruiter

* Register / Login
* Job post karna
* Applicants dekhna
* Candidate select/reject karna
* Chat with candidates

## 3.3 Admin

* Dashboard access
* Users manage (ban/unban)
* Jobs manage (delete/edit)
* Recruiters manage
* Reports & analytics
* System monitoring

---

# 4. 🚀 Features (Phase Wise)

---

## 🔹 Phase 1: Authentication & Basic Setup

### Features:

* User Registration (User / Recruiter)
* Login System (JWT based)
* Role-based Authentication
* Password Encryption (bcrypt)

### Deliverables:

* Auth APIs
* Login/Register UI

---

## 🔹 Phase 2: User Profile System

### Features:

* Profile creation (name, skills, bio)
* Resume upload (PDF)
* Profile picture upload
* Edit profile

### Deliverables:

* Profile APIs
* Profile UI

---

## 🔹 Phase 3: Job Management System

### Features:

* Recruiter job post karega
* Job edit/delete
* Job listing page
* Job search (filters):

  * Location
  * Salary
  * Skills

### Deliverables:

* Job APIs
* Job listing UI

---

## 🔹 Phase 4: Application System

### Features:

* User job apply karega
* Application status:

  * Pending
  * Accepted
  * Rejected
* Recruiter applicants dekhega

### Deliverables:

* Application APIs
* Application UI

---

## 🔹 Phase 5: Admin Panel

### Features:

* Dashboard (stats)
* Manage users
* Manage jobs
* Manage recruiters
* Delete/report content

### Deliverables:

* Admin APIs
* Admin Dashboard UI

---

## 🔹 Phase 6: Chat System (Real-time)

* Chat APIs

## 🔹 Phase 7: Advanced Features (2026 Level 🔥)
